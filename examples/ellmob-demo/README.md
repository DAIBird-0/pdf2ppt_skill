# ELLMob Demo

This folder contains a public-paper demo deck and preview image used to show the skill's output style.

The demo references:

- `ELLMob: Event-Driven Human Mobility Generation with Self-Aligned LLM Framework`
- Listed in the deck as an ICLR 2026 conference paper by Yusong Wang et al.

The deck demonstrates slide structure, figure mapping, academic visual style, and per-slide speaker notes. The paper content is not owned by this repository and is not relicensed under MIT. See `../../THIRD_PARTY_NOTICES.md`.
